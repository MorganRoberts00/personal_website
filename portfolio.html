Portfolio
I was given the task of designing functional implementations of abstract-data-types. These included: Linked-Lists (Single and Double), Binary-Search-Trees (AVL & Black-Red), Stacks, Queues, and Hash-Maps (separate-chaining & open-addressing). These projects were intensive but highly rewarding because they gave me a greater understanding of the lower level mechanisms of software as well as how to avoid common issues.
I even designed my own website once! When I started, I knew next to nothing about web-development nor the tools required to implement a website. Over the span of 2 weeks, I was able to learn the necessary languages such as html, css, and javascript, as well as their functions. I was also able to learn Bootstrap , which is an open-source css framework. I highly recommend using it for anyone curious. The image of the webpage is also a link to my website's github repository. Go ahead and stop by if you feel so inclined.
